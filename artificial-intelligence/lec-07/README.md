# lec-07 — Artificial Neural Networks — classification

Train a feed-forward neural network to classify images of clothing (Fashion-MNIST).

## Objectives

- Build and train a small neural network
- Understand layers, activations, loss and epochs
- Evaluate predictions on held-out images

## Dataset

Fashion-MNIST (downloaded by the notebook)
