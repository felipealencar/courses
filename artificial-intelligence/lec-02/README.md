# lec-02 — Genetic Algorithms (bio-inspired optimization)

An evolutionary search that breeds a population of candidate solutions toward an objective.

## Objectives

- Represent candidate solutions as a population
- Apply selection, crossover and a fitness function
- Watch a population converge over generations

## Dataset

`beetles.csv` (illustrative population, generated in the script)

## Files

- `beetle_genetic_algorithm.py` — runnable script
- `beetles.csv` — sample data

## Notes

Run with: `python beetle_genetic_algorithm.py`
