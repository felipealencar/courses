# lec-08 — Time-series forecasting with a neural network

Forecast tomorrow's temperature in Maceió with lag features and an MLP, against a persistence baseline.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/felipealencar/courses/blob/artificial-intelligence/artificial-intelligence/lec-08/time_series_forecasting.ipynb)

## Objectives

- Turn a time series into a supervised problem with lag features
- Use a chronological (non-shuffled) split
- Compare an MLP to a naive baseline with MAE/RMSE
