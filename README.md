# Introdução a Redes de Computadores

> Disciplina ministrada pelo **prof. Felipe Alencar** no
> **IFAL — Campus Arapiraca**.
> Esta branch faz parte do repositório [`courses`](https://github.com/felipealencar/courses)
> (organizado por branches). Veja o [índice de disciplinas](https://github.com/felipealencar/courses/tree/master).

Fundamentos de redes de computadores: arquitetura em camadas, protocolos e os
modelos de referência que sustentam a comunicação na Internet.

## Ementa

- Conceitos e topologias de redes.
- Modelos de referência **OSI** e **TCP/IP**.
- Camadas de aplicação, transporte, rede e enlace.
- Endereçamento IP e roteamento.
- Principais protocolos (HTTP, DNS, TCP/UDP, IP).

> para anotações complementares.
