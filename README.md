# PHP Avançado

> Disciplina ministrada pelo **prof. Felipe Alencar** no
> **IFAL — Campus Arapiraca**.
> Esta branch faz parte do repositório [`courses`](https://github.com/felipealencar/courses)
> (organizado por branches). Veja o [índice de disciplinas](https://github.com/felipealencar/courses/tree/master).

Tópicos avançados de desenvolvimento em **PHP** com o framework **Laravel**,
com ênfase em qualidade e testes automatizados.

## Ementa (conteúdo aula a aula)

| Aula | Tema | Destaques |
| --- | --- | --- |
| `lec-02` | Testes automatizados com PHPUnit em Laravel | projeto `phpunit-tests`, testes de unidade e feature |

## Pré-requisitos

- PHP e orientação a objetos (ver disciplina
  [`php-basic`](https://github.com/felipealencar/courses/tree/php-basic)).
- Composer instalado.

## Ambiente e execução

```bash
cd php-advanced/lec-02/phpunit-tests
composer install
php artisan test     # ou ./vendor/bin/phpunit
```

## Como obter o código

```bash
git clone -b php-advanced git@github.com:felipealencar/courses.git
```
