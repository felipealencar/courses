# JavaScript Avançado

> Disciplina ministrada pelo **prof. Felipe Alencar** no
> **IFAL — Campus Arapiraca**.
> Esta branch faz parte do repositório [`courses`](https://github.com/felipealencar/courses)
> (organizado por branches). Veja o [índice de disciplinas](https://github.com/felipealencar/courses/tree/master).

JavaScript além do navegador: visualização de dados, execução no servidor com
**Node.js** e construção de aplicações web seguindo o padrão **MVC**.

## Ementa (conteúdo aula a aula)

| Aula | Tema | Destaques |
| --- | --- | --- |
| `lec-01` | Visualização de dados no front-end | Chart.js, Bootstrap, leitura de CSV |
| `lec-02` | Servidor web com Node.js e Express | servidor HTTP, rotas, arquivos estáticos |
| `lec-03` | Aplicação web em arquitetura MVC | Express + EJS, controllers, models, routes e views |

## Pré-requisitos

- HTML, CSS e JavaScript básico (ver disciplina
  [`pweb-html-css-js`](https://github.com/felipealencar/courses/tree/pweb-html-css-js)).
- Node.js e npm instalados.

## Ambiente e execução

Cada aula é um projeto Node independente. Entre na pasta da aula, instale as
dependências e execute:

```bash
cd javascript-advanced/lec-03
npm install
node app.js
```

## Como obter o código

```bash
git clone -b javascript-advanced git@github.com:felipealencar/courses.git
```

Você também pode baixar apenas o diretório de uma aula usando o
[Download Directory](https://download-directory.github.io/).
