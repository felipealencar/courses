# Processo de Desenvolvimento de Software

> Disciplina ministrada pelo **prof. Felipe Alencar** no
> **IFAL — Campus Arapiraca**.
> Esta branch faz parte do repositório [`courses`](https://github.com/felipealencar/courses)
> (organizado por branches). Veja o [índice de disciplinas](https://github.com/felipealencar/courses/tree/master).

Processos e práticas de engenharia de software: ciclo de vida, metodologias de
desenvolvimento, controle de versão com **Git/GitHub** e desenvolvimento de
aplicações de exemplo.

## Ementa

- Modelos de processo de software e metodologias ágeis.
- Versionamento com Git e fluxo de trabalho com branches e _pull requests_.
- Desenvolvimento de aplicação de exemplo (projeto `laravel-tasks`).
- Atividade prática com aplicação em **Kotlin** (`lec-05`).

## Atividade prática (`lec-05`)

1. Clonar o repositório da disciplina.
2. Fazer o _switch_ para a branch `software-development-process`.
3. Criar um novo ramo baseado nessa branch.
4. Implementar uma aplicação de exemplo em Kotlin.
5. Enviar o código e abrir um _pull request_.

## Como obter o código

```bash
git clone -b software-development-process git@github.com:felipealencar/courses.git
```
